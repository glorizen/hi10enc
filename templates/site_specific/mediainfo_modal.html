Help on MediaInfo

MediaInfo is a metadata utility that allows user to see the details of targeted file. For example, for a file having video and audio MediaInfo can show the details about video stream and audio stream in depth.
Using this utility you can send your file's details here and in return get an auto-generated commands to encode the files on your system.

At the time being, only XML formatted information can be parsed successfully. Hence, given that you have mediainfo installed on your system, you can get your file details in XML format by typing:

mediainfo --output=XML your_filename.mp4
