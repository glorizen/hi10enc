About the Web Application

This application is supposed to act as a stepping stone for noobs to get started on their encoding adventure. It relies heavily on ffmpeg since it provides all functionalities in a single package. Users don't need to install any other applications to encode things.

Muxing, which is combining the data streams like video, audio, subtitle etc., can be done by ffmpeg to some extend but for the time being its muxing is a bit buggy and can't be trusted for reliable playback. Hence, users will have to use mkvmerge for the muxing part which is more-or-less the last part anyway.
